# IPL EDA Project

## Overview

This project performs Exploratory Data Analysis (EDA) on an IPL dataset using Python.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Analysis Performed

* Data Loading
* Data Cleaning
* Missing Value Analysis
* Duplicate Removal
* Data Visualization
* Correlation Analysis
* Outlier Detection

## Key Insights

* Identified patterns in IPL match data.
* Analyzed team performance trends.
* Explored relationships between variables using visualizations.
